## Conventions

- **Story IDs**: `IB-NN` (InkBytes-NN), continuous across epics, starting at IB-12 (IB-1..IB-11 are Sprint-1 INK-*).
- **Story points**: Fibonacci (1, 2, 3, 5, 8, 13).
- **Sprint capacity assumed**: 30 pts per 2-week sprint for a team of 2 engineers.
- **Risk links**: References to `quality/risk-register.md` IDs.
- **ADR links**: References to existing ADRs or proposed (in ADR-index).
- **Language**: stories in English (developer-friendly, paste-ready for Linear/Jira); strategy docs in Spanish.
