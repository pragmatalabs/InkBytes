How it works

A reproducible pipeline, not a wrapper over a chatbot.

Five stages turn the firehose of the open web into one trustworthy page — with a hard rule at the center.

01

Collect

Harvest many trusted outlets in parallel, in real time, across EN and ES.

02

Enrich

Tag the people, organizations, places, topic and language; summarize each article.

03

Cluster

Group articles on the same event by meaning, not keywords.

04

Synthesize

Write one balanced page once ≥ 2 sources agree it's a story. Every claim cited.

05

Verify

A QA pass drops any claim that can't be tied back to a real source.

The ≥ 2-source rule Single-source rumors never become an InkBytes page. If only one outlet has it, we wait.
